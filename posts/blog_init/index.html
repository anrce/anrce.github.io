hugo下载

https://github.com/gohugoio/hugo

下载的是 hugo_extended版本，支持 scss/sass等语法

博客初始化

使用的是LoveIt主题

1
2
3
4
5

hugo new site blog
cd blog
git init
git submodule add https://github.com/dillonzq/LoveIt themes/LoveIt
echo "theme = 'LoveIt'" >> hugo.toml

新版hugo v0.136.2使用loveit主题报错修复

ERROR deprecated: hugo.IsMultilingual was deprecated in Hugo v0.124.0 and will be removed in Hugo 0.137.0. Use hugo.IsMultilingual instead.

替换相关文件的内容

1
2
3

.Sites.Default -> .Sites.Default
hugo.IsMultilingual -> hugo.IsMultilingual
.Site.Author -> .Site.Params.author
